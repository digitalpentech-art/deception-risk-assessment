# Multimodal Deception Risk Assessment System

This system is an assistive behavioral risk assessment tool designed for research and investigative support. It is NOT a lie detector.

## Roadmap Overview
- Project encompasses 17 phases, from Requirements Analysis to Documentation.
- Features: Facial, Speech, Voice, Micro-expression analysis and multimodal fusion.
- Technology Stack: Python, TensorFlow, Flask, HTML/JS.
