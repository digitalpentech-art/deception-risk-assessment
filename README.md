# Multimodal Deception Risk Assessment System

## Overview
Design and Implementation of a Multimodal Deep Learning System for Deception Risk Assessment During Suspect Interrogation Using Facial Expressions, Speech Patterns, and Voice Analysis.

## Project Goal
Develop a multimodal AI system for behavioral risk assessment (Low, Medium, High Risk).
- Captures facial expressions, speech emotion, voice stress, and micro-expressions.
- Fuses modalities via Transformers for risk estimation.

## Ethical Notice
**IMPORTANT:** This AI system is an assistive behavioral risk assessment tool designed for research and investigative support. It is **not** a lie detector and should not be used as sole evidence in legal decision-making.

## Technology Stack
- Python
- TensorFlow/Keras
- Flask
- HTML/JS
